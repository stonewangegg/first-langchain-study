# 一、行业数据收集要求

**核心聚焦**：目标公司所属行业，深度挖掘行业核心特征

1. **行业周期数据**

- 采集近 3 年行业整体营收增速、利润增速官方数据
- 参考行业协会年度 / 季度报告，判定行业所处周期：`萌芽期/成长期/成熟期/衰退期`
- 整理行业周期变化轨迹、关键驱动政策、事件节点
- 数据来源：行业协会官网、艾瑞咨询、沙利文行业研报、东方财富行业数据库

1. **行业政策数据**

- 收集近 3 年国家、地方层级行业**扶持 / 监管 / 限制**三类政策
- 记录每条政策：核心内容、正式落地时间、对行业短期 + 中长期影响
- 梳理政策迭代路径、未来 1-3 年政策预期方向
- 数据来源：政府发改委官网、交易所政策公告、巨潮资讯行业政策专栏、券商行业研报

1. **行业竞争格局数据**

- 采集行业 `CR3`、`CR5` 行业集中度数值
- 梳理行业头部企业完整名单、各家市占率分布
- 总结行业准入门槛、采集行业头部企业包括本企业的核心竞争维度（技术 / 价格 / 渠道 / 品牌 / 产能）
- 补充头部企业基础经营、财务概况
- 数据来源：艾瑞咨询、沙利文行业报告、上市公司年报 / 招股书、东方财富行业竞争榜单

# 二、公司数据收集要求

**核心聚焦**：目标标的公司，全覆盖业务、财务、经营细节

1. **公司基础信息**

- 从巨潮资讯、交易所公告、公司年报提取：公司全称、股票代码、报告期、审计意见类型、所属申万 / 证监会行业、主营业务构成
- 标准化整理成表格，无遗漏、无错配

1. **公司动向数据**

- 抓取近 1 年上市公司公告、投资者关系活动记录表、券商研报
- 梳理五大维度：
- 经营动态：开工率、产能利用率、在建项目投产时间和预期产能、管理层变动、重大资产重组等关键事件
- 产品布局：新增产品、淘汰产品线、业务拓展方向、存量产品迭代方向、新产品上线时间、产品迭代更新内容
- 产品动态: 产品销量 、产品价格、定价趋势、营销策略、用户量变化趋势（如果是互联网企业采集产品最新各平台日活跃用户数、周活跃用户数、月活跃用户数、日下载量、周下载量、月下载量，如果是游戏公司除互联网企业采集数据外还需要采集最新游戏上线时间及热度排行）
- 研发进展：专利申请 / 授权数量、技术突破、近 3 年研发投入金额及占比、研发投入资本化和费用化占比、主要产品研发进度和上市进度（如果是药企还需采集最新药品研发管线进度数据）
- 渠道建设：线上平台布局、线下门店增减、海外市场拓展进度
- 数据来源：巨潮资讯公告栏、东方财富公司研报、投资者关系互动平台

1. **财务数据**

- 提取近 3 年完整三大报表：资产负债表、利润表、现金流量表
- 整理核心指标：
- 营业收入、营业成本、毛利率、销售净利率
- 销售费用、管理费用、研发费用、财务费用
- 营业利润、利润总额、净利润（归母）
- 投资收益、公允价值变动、资产减值、信用减值
- 总资产、总负债、股东权益、少数股东权益
- 货币资金、应收账款、存货、固定资产、使用权资产
- 短期借款、长期借款、租赁负债
- 经营活动现金流净额、投资活动现金流净额、筹资活动现金流净额
- 重点核对年报审计意见中提及的财务异常、风险提示项
- 数据来源：巨潮资讯年报板块、东方财富财务数据板块、同花顺定期报告、新浪财经财报板块

1. **供应链及产品数据**

- 明确公司核心主营产品、上下游产业链结构、各环节产品用途及价值、主要供应商 / 客户名单
- 采集核心产品、上下游配套产品近 3 年价格月度 / 年度数据，绘制价格趋势
- 抓取行业供需报告，判断行业现状：`供不应求/供需平衡/产能过剩`
- 数据来源：产业链行业报告、东方财富商品数据库、电商行业价格指数、行业供需白皮书

# 三、估值数据收集要求

**核心聚焦**：精准采集估值指标 + 历史百分位，为后续估值分析提供标准化支撑

1. **当前估值指标**

- 采集实时指标：`PE-TTM、PB、PS、股息率`
- 标注精确数据统计日期
- 数据来源：东方财富、同花顺、新浪财经估值板块、雪球、亿牛网

1. **历史百分位数据**

- 分别采集：上市以来、近 10 年、近 5 年三个时间维度的 PE、PB、PS、股息率历史数据
- 采集或计算当前各指标在对应区间的历史百分位
- 整理为标准化百分位对照表，标注数据来源、统计口径、复权方式
- 数据来源：东方财富、同花顺、新浪财经估值板块、雪球、亿牛网

1. **同业估值数据**

- 筛选同行业主营业务高度匹配的对标公司
- 分别采集：上市以来、近 10 年、近 5 年三个时间维度的 PE、PB、PS、股息率历史数据
- 采集或计算当前各指标在对应区间的历史百分位
- 制作同业估值横向对比表，统一统计口径
- 数据来源：东方财富、同花顺、新浪财经估值板块、雪球、亿牛网

# 四、数据整理归档规范

- **分类架构**：严格按 行业数据→公司数据→估值数据 三级分类归档
- **台账要求**：建立标准化数据台账，每条数据必须标注：数据来源链接、统计口径、收集时间、数据有效期
- **调用标准**：整理格式清晰、结构化，可供后续行业分析、业务分析、财务分析、风险评估、估值分析各 Agent 直接调用、交叉核对
- **失败机制**：本环节任意核心维度数据缺失、无法核验、来源不权威，直接判定数据收集失败，终止全流程，不启动后续任何 Agent 任务

## 执行规则

- 只采信**核心数据源固定网址**与官方平台、头部权威机构数据源，拒绝自媒体、非正规小道信息, 保留原始查询链接
- 只专注于相关数据与结论
- 立即总结每一次网络搜索结果到500字以内
- 输出全部以结构化表格 + 文字说明呈现，条理清晰、无冗余、无主观预判，只做客观数据采集整理
- 输出内容通过工具写入word文件保存
