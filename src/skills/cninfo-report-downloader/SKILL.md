# 巨潮报告下载器

## 功能说明

该技能用于从巨潮资讯网(www.cninfo.com.cn)下载指定公司、指定年份的年报和季度报告。支持下载PDF格式的报告文件。

## 调用方法

当用户需要获取上市公司的财务报告时，使用此技能。用户需要提供以下信息：

1. 公司名称或股票代码
2. 报告年份
3. 报告类型（年报或季度报告）
4. 季度（仅季度报告需要）

## 使用示例

**示例1：下载年报**
用户：请下载贵州茅台2023年的年报
系统：正在从巨潮网站下载贵州茅台2023年的年报...

**示例2：下载季度报告**
用户：请下载腾讯控股2024年第一季度的报告
系统：正在从巨潮网站下载腾讯控股2024年第一季度的报告...

## 实现原理

1. 通过公司名称或股票代码在巨潮网站搜索公司信息
2. 获取公司的A股代码和公司ID
3. 根据指定年份和报告类型构建搜索URL
4. 解析搜索结果，找到对应的报告链接
5. 下载PDF报告文件到当前工作目录

## 注意事项

1. 巨潮网站可能会有访问限制，建议合理控制请求频率
2. 部分公司可能没有公开某些年份的报告
3. 下载速度取决于网络状况和文件大小
